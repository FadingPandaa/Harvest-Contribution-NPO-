1. PROJECT SUMMARY
------------------
Harvest Contribution is a non-profit web platform focused 
on fighting hunger through community gardens. Built with 
HTML5, CSS3, and JavaScript, it features dynamic donation 
processing, user authentication, and interactive galleries 
to coordinate volunteer efforts and sustainable agricultural 
education for vulnerable communities.

2. PROJECT COMPONENTS
---------------------
The project is organized into three primary layers:

A. STRUCTURE (HTML5)
* index.html: Homepage with mission statement and goal tracking.
* about us.html / services.html: Organizational history and program details.
* gallery.html: Visual portfolio of community impact.
* login.html: Portal for member and volunteer access.
* contact us.html: Communication channel for public inquiries.
* Donations & Sign Up.html: Interactive support and registration hub.
* thank you... .html: Confirmation responses for user actions.

B. DESIGN (CSS3)
* Style.css: 
  The master stylesheet controlling the visual identity, 
  responsive flexbox layouts, navigation aesthetics, and 
  lightbox UI components.

C. FUNCTIONALITY (JavaScript ES6+)
* lightbox.js: Powers the interactive image viewing experience.
* login and sign up.js: Validates and manages user accounts.
* Enter Amount.js: Logic for handling custom donation values.
* Email Sign Up.js: Manages newsletter and registration data.
* Copywrite and Year.js: Dynamically updates legal footer info.

D. DOCUMENTATION
* PoE Web Development (Full).pdf: 
  Complete project documentation covering research, planning 
  phases, target audience analysis, and implementation steps.

3. HOW TO RUN
-------------
1. Maintain the directory structure (HTML in root, CSS in /CSS, 
   JavaScript in /JavaScript).
2. Open 'index.html' in any modern web browser.
3. Ensure JavaScript is enabled to allow the donation logic 
   and gallery effects to function.

4. TECHNICAL SPECIFICATIONS
---------------------------
* Languages: HTML5, CSS3, JavaScript.
* Features: Client-side validation, Dynamic DOM manipulation, 
  Responsive Design, and Event-driven UI.
